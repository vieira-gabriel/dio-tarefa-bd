# Desafio de criação para matéria Fundamento de Dados

Repositório com a tarefa referente a matéria de banco de dados

# Intenção

Quero que a IA analise comentários de clientes de companhias aéreas para identificar as principais reclamações e pontos de melhoria.
O resultado será usado por analistas de dados juntos com um time de marketing para apoiar melhorias no atendimento e nas soluções de problemas.
A entrega deve conter um resumo executivo, uma tabela com os principais temas, exemplos de comentários, lista de insites e recomendações práticas.
O resultado será considerado bom se for claro, organizado, baseado nos comentários reais fornecidos e útil para priorizar ações de melhoria.

# Contexto e restrições

## Contexto
Estou trabalhando com feedbacks de clientes de companhias aéreas relacionados a atrasos, mal atendimentos, overbook, falta de informações por parte das companhias.

## Dados disponíveis
A base contém data do comentário, canal de atendimento, companhia aérea, texto do feedback, produto citado e nota de satisfação de 1 a 5.

## Critérios de análise
A IA deve classificar os feedbacks por temas, urgência, canal e categoria da reclamação (overbook, atraso, mal atendimento,...).

## Cuidados e restrições:

Use apenas os dados reais encontrados em sites de reclamação ou da própria companhia.
Não invente números, causas ou conclusões.
Não exponha dados pessoais ou sensíveis.
Se houver informação insuficiente, indique a limitação.
Use linguagem simples, clara e direta.
